# Inventory Management and Financial Analysis System

Desktop application developed in C# using Windows Forms and SQL Server for inventory management and financial analysis.

Features:
- Product management (CRUD operations)
- Inventory tracking and categorization
- Financial calculations:
  - EOQ (Economic Order Quantity)
  - Reorder Point
  - ABC Classification
- Data visualization using DataGridView
- Export reports to PDF and PNG

Technologies:
- C# (.NET, Windows Forms)
- SQL Server (Stored Procedures)
- iTextSharp (PDF generation)

This project was developed as part of an academic assignment focused on applying data analysis and inventory management concepts.
